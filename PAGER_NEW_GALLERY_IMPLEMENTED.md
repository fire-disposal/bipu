## 🎯 集成方式

### 导航流程
```dart
// new_pager_page.dart
void _navigateToGallery(BuildContext context) {
  Navigator.of(context).push(
    MaterialPageRoute(
      builder: (context) => const NewOperatorGalleryView(),
    ),
  );
}
```

### 状态获取
```dart
final vm = context.watch<PagerVM>();
final allOperators = vm.operatorService.getAllOperators();
final unlockedCount = allOperators.where((op) => op.isUnlocked).length;
```

## ✨ 完整流程

### 用户流程
```
1. 点击"接线员图鉴" 按钮 (AppBar)
   ↓
2. 导航至 NewOperatorGalleryView
   ↓
3. 显示统计信息 (解锁进度)
   ↓
4. 显示接线员网格
   - 已解锁: 显示立绘 + 名字
   - 未解锁: 显示黑影 + 问号
   ↓
5. 用户点击已解锁卡片
   ↓
6. 显示详情弹窗
   - 立绘图片
   - 名字 + 标签
   - 描述文案
   - 统计信息
   - 关闭按钮
   ↓
7. 点击关闭返回列表
```

### 数据流
```
PagerVM.operatorService
  ↓
getAllOperators() → List<OperatorPersonality>
  ↓
过滤: isUnlocked / 全部
  ↓
构建卡片网格
  ↓
点击 → 显示详情
```
